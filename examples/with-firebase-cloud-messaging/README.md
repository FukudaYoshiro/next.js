## How to run

Install it and run:

```bash
npm install
npm run build
npm run dev
```

## Set your send id

set your `messagingSenderId` in `static/firebase-messaging-sw.js` and `utils/webPush.js`

Deploy it to the cloud with [now](https://zeit.co/now) ([download](https://zeit.co/download))

```bash
now
```
